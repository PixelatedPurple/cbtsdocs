What do we use Github for?

We use GitHub to not only store our code but some Services like Vercel Require a GitHub repository to edit and deploy web applications.
We also use it to make and keep our projects open sourced so everyone can see how we do what we do anytime anywhere.
Another use we have us for easy team collaboration. Yeah most online IDE and compilers have a multiplayer feature fir editing but we like to use github because we can deploy all of our changes at once. It's a great way to collaborate because you can read over your work or you teammates work for errors and flaws before production.
